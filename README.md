# Malay-consumer-complaints-classification

Classify 5000 samples of consumer complaints in banks. These are classified as one of 5 classes below. 

    Mortgage,                       
    Checking or savings account,  
    Credit card,                   
    Bank account or service,         
    Consumer Loan,
